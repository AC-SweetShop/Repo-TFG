IPyWidgets¶

IpyWidgets es una librera que nos permite importar widgets FrontEnd para poder interactuar con las grficos. Podemos invocarlo con interact.

In [12]:
#creamos la funcion grafico varible para comparar la columna 1 seleccionable desde el ComboBox con el Valor Mediano
@interact(col1=df.columns.tolist())
def grafico_variable(col1):
    df.plot.scatter(x=col1, y="VALOR_MEDIANO")
    plt.title("{} vs VALOR_MEDIANO".format(col1))

BokehJS¶

BokehJS es una librera que nos permitir realizar graficas pensadas para mostrar grficos en un navegador. Bokeh es una librera para visualizaciones interactivas diseada para funcionar en los navegadores web modernos. Su objetivo es proporcionar una construccin elegante y concisa de grficos modernos al estilo de D3.js, y para ampliar esta capacidad con la interactividad y buen rendimiento sobre grandes volmenes de datos. Bokeh puede ayudar a cualquier persona a crear en forma rpida y sencilla grficos interactivos, dashboards y aplicaciones de datos
